# UX tecnico mobile-first

## Principios

- Pantalla inicial centrada en trabajos de hoy.
- Acciones primarias siempre visibles.
- Botones grandes.
- Formularios cortos.
- Texto guardado en borrador local.
- Acceso rapido a Maps, Waze y llamada.
- Codigo interno del equipo visible para trazabilidad.

## Pantallas

### Dashboard tecnico

- Conteo de avisos y revisiones.
- Siguiente aviso destacado.
- Avisos urgentes.
- Lista de avisos.
- Lista de revisiones.
- Partes abiertos.

### Detalle aviso

- Prioridad.
- Cliente.
- Instalacion.
- Equipo.
- Codigo interno del equipo.
- Descripcion de averia.
- Botones Maps, Waze, llamar e iniciar parte.
- Instrucciones de acceso.
- Historial rapido del equipo.

### Parte de trabajo

- Trabajo realizado.
- Observaciones.
- Resultado.
- Hasta tres lineas rapidas de materiales.
- Fotos desde camara.
- Firma.
- Cierre del parte.

### Firma cliente

- Resumen del trabajo.
- Nombre del firmante.
- Canvas grande para firma tactil.
- Limpiar / guardar.
